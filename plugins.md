# インストール済み Claude Code プラグイン

## インストール方法
```
claude plugin install <plugin-name>
```

## 一覧

| プラグイン | 用途 |
|-----------|------|
| `frontend-design` | AIっぽくない本番品質のフロントエンドデザイン生成 |
| `playwright` | ブラウザ自動テスト（UI動作確認） |
| `claude-code-setup` | コードベース分析 → hooks/skills/MCP等の自動化設定を提案 |
| `skill-creator` | スキルの新規作成・改善・パフォーマンス評価（skill-makerの代替） |
| `plugin-dev` | プラグイン開発総合ツールキット（hooks/MCP/構造設計/コマンド/エージェント） |
| `claude-md-management` | CLAUDE.mdの作成・管理・最適化支援 |
| `security-guidance` | コード編集時にセキュリティリスク（XSS, eval, pickle等）を検知してwarningを出すhook |
